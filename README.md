# Build-Web-Server

Create Web Server using NGINX

Finish project with this step :

1. Clone page with git bash through this command: $git clone https://github.com/dicodingacademy/a387-jarkom-labs.git

2. Enter the directory: cd a387-jarkom-labs/

3. Go to Ubuntu Terminal and enter directory (a387-jarkom-labs)

4. $sudo service nginx status to check nginx status

5. cat /etc/nginx/sites-available/default to check configuration

6. sudo nano to edit using txt command

7. edit app.js

8. restart nginx

9. Start npm

10. Open browser using public IP
